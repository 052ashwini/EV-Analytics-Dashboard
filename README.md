# 🚀 Electric Vehicle Analytics Dashboard

## 📊 Overview
This project is a Power BI dashboard analyzing Electric Vehicle data.

## 🎯 Objectives
- EV growth trend analysis
- EV type distribution (BEV vs PHEV)
- Top manufacturers
- Electric range comparison
- Geographic distribution

## 🛠 Tools Used
- Power BI
- DAX
- Data Cleaning
